# Space Blasters — Math Mission

Voice-answered, math-powered space shooter for kids. Adding, subtracting, multiplying,
dividing, and missing-number (algebra) problems fall from the sky — say the answer out loud
to blast them. Cross-device leaderboard (name + 4-digit PIN) via Supabase.

The whole game is `index.html` (self-contained, no build step). To deploy, see
**DEPLOY_WITH_CLAUDE_CODE.md**. Run locally by opening `index.html` in Chrome or Edge.
