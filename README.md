# Space Blasters — Math Mission

Voice-answered, math-powered space shooter for kids. Adding, subtracting, multiplying,
dividing, and missing-number (algebra) problems fall from the sky — say the answer out loud
to blast them. Cross-device leaderboard (name + 4-digit PIN) via Supabase.

The whole game is `index.html` (self-contained, no build step). Run locally by opening
`index.html` in Chrome or Edge. The game is already live in production — do not re-run any
setup/deploy steps.

This repo also hosts the **Veritas Learning Hub / Academy OS** build. Its source of truth is
`docs/SPEC.md`, governed by `docs/ONE_PAGE.md` (see `CLAUDE.md`).
